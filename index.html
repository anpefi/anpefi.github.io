Hi! I’m Andrés Pérez-Figueroa, a postdoctoral researcher in the Phylogenomics Lab at the University of Vigo. My current research here is about population genetics of tumoral cells. My background is related to conservation and population genetics as a computational biologist. I can write code (not very well) to simulate and analyze biological data for understanding its evolutionary implications.

Some random thoughts

Redesigning the website

2017-12-20 2 min read
I’ve changed the website design. For the last 2 years this site was relying on the Jekyll framework using the Minimal Mistakes template. Recently, I started to dive deeper in Rstudio, rmarkdown and the tidyverse ecosystem. Few days ago I discovered in Twitter about the new Yihui Xie’s package called blogdown that allows to create static websites using Rmarkdown and Hugo. After a quick read of the Xie’s book about this package I decided to change this website from Markdown+Jekyll to Rmarkdown+Blogdown+Hugo.
